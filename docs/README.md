# Nuri 개발 문서

이 디렉터리는 각 애플리케이션의 현재 구현 상태와 개발 방법을 설명한다.

- [Web](./web/README.md): 화면, 라우트, 클라이언트 상태와 API 연동 상태
- [API](./api/README.md): 인증, 사례·상담 API, 데이터 모델과 migration
- [AI worker](./ai-worker/README.md): 내부 AI 엔드포인트, 요약·슈퍼비전과 실행 방법

문서는 계획보다 현재 코드의 동작을 우선해서 기록한다. 코드와 문서가 다르면 코드 변경과 함께 관련 문서를 갱신한다.
